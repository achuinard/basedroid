Basedroid

Framework for New JSON-based Android Apps

Download Download fork Basedroid on GitHub

About

I've found that whenever I want to make an Android app that interacts with some API over JSON, I was rewriting the same boilerplate code every time. There were also some other pieces of Android boilerplate, such as action bars and state managers, for which I wanted to create a reusable base. The project also aims to keep Android permissions to a bare minimum - currently it only uses the network request permission.

This is Basedroid.

$ git clone git://github.com/achuinard/basedroid.git
$ cd basedroid
$ cat README
$ ant clean debug

Roboguice

Basedroid includes roboguice, a dependency injection framework used by popular apps such as Facebook Messenger and Google Docs.

BasedroidHttpClient

A singleton HTTP client, wrapped around Apache HttpClient, which provides a simple method to make GET / POST requests. Examples of deserializing JSON into POJOs (plain old Java objects) are included.

StateManager

Another singleton which allows you to store basic key-value pairs. I have found that the use of SQLite is unnecessary in apps that I have developed. The StateManager is wrapped around the Android SharedPreferences class and includes some magic that serializes data into JSON before saving it.

Action Bar

Basedroid includes a reusable action bar, a UI pattern now recommended by Google for all Android apps. You can use the action bar to eliminate the need for the menu button.

Examples

You will find several examples inside Basedroid - ListViews, asynchronous HTTP requests, state management, and more.
